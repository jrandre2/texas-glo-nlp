# Visualizations

This folder contains **dashboards and Sankey visuals** that can be opened directly (no code required).

## Most-used files

- `harvey_dashboard_standalone.html` — primary interactive dashboard
- `harvey_extended_dashboard.html` — additional drill-down views
- `harvey_sankey_5b.png` / `harvey_sankey_57m.png` — slide-friendly Sankey images
- `harvey_sankey_recipients.png` — recipients Sankey image

## Notes

- The `.png` files are the easiest to share in slides.
- The `.html` dashboards work best when opened in Chrome.
